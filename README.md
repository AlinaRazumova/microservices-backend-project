# microservices-backend-project

Academic project: microservices-based system with REST API, authentication, database, Docker, and web/mobile client.

We have successfully shifted from the planning phase to active development. The backbone of the ecosystem is now functional.

### New:
*   **Auth Service**: Completed. Handles secure user registration, login, and JWT issuance. 
*   **Task Service**: Completed. Core business logic for managing tasks, statuses, and assignments is live. 
*   **Infrastructure**: Services are containerized and ready for orchestration.

## Tech Stack (Current)
*   **Architecture:** Microservices
*   **Auth:** JWT / Spring Security 
*   **Communication:** REST API
*   **Deployment:** Docker & Docker Compose
